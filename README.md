# Tab Re(gex)name

This is a smart tab renamer. You can use regex matching on title for specific URLs to rename tab titles at will.

## For Chrome (Manifest V3)

Uploading extensions to the Chrome Store means paying a 5$ fee, so I won't do that.

The extension in this branch is adapted to Manifest V3, so it is usable in Chrome.
